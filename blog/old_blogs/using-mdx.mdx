该主题附带在 `astro.config.ts` 配置文件中安装和配置的 [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) 集成。如果您不想使用 MDX，可以通过从配置文件中删除集成来禁用支持。

## 为什么选择 MDX？
MDX 是 Markdown 的一种特殊风格，支持嵌入式 JavaScript 和 JSX 语法。这解锁了将 JavaScript 和 UI 组件混合到 Markdown 内容中的能力，例如交互式图表或警报等。

如果您已有在 MDX 中创作的内容，这种集成有望使迁移到 Astro 变得轻而易举。
